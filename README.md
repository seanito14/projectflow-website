# ProjectFlow

ProjectFlow is a premium, locally-hosted project management tool designed and built for the **Mistral Worldwide Hacks**.

It features a beautiful glassmorphic UI, lightning-fast vanilla Javascript local storage, and an intuitive Kanban board for tracking your work.

## Features

* **Local Storage Persistence**: All projects and tasks are saved securely in your browser's local storage.
* **Kanban Board**: Drag-and-drop tasks between "To Do", "In Progress", and "Done" states.
* **Premium UI**: Designed with smooth animations, neon accents, and a frosted glass aesthetic.
* **Completely Serverless**: No backend required. Simply serve the static files and you're good to go.

## Quick Start (Local)

1. Clone the repository.
2. Run the included serve script:

   ```bash
   ./serve.sh
   ```

3. Open `http://localhost:8000` in your browser.
